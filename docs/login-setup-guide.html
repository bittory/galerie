1. Firebase 설정 (Google 로그인 활성화)

01

Firebase 프로젝트 생성

아직 프로젝트가 없다면:

  • console.firebase.google.com 접속
  • 프로젝트 추가 → 프로젝트 이름 입력 (예: galerie-app)
  • Google Analytics는 선택사항 (비활성화 가능)
02

웹 앱 등록 및 설정값 복사

프로젝트 설정 → 내 앱 → 웹 앱 추가 (</> 아이콘) → 설정값 복사

const FIREBASE_CONFIG = {
  apiKey: "AIza...",      // ← 여기 실제 값 입력
  authDomain: "your-project.firebaseapp.com",
  projectId: "your-project-id",
  storageBucket: "your-project.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123:web:abc"
};

위 값을 docs/index.html 상단 FIREBASE_CONFIG 객체에 붙여넣기

03

Authentication 공급자 활성화

Firebase 콘솔 → Authentication → Sign-in method → 아래 공급자 활성화:

  • ✅ Google — 프로젝트 지원 이메일 입력 후 저장
  • ✅ 이메일/비밀번호 — 활성화만 하면 됨
  • ✅ GitHub (선택) — 클라이언트 ID/Secret 필요 (아래 4번)
04

승인된 도메인 등록

Firebase 콘솔 → Authentication → Settings → 승인된 도메인 → 아래 추가:

bittory.github.io
localhost
중요: 이 도메인이 없으면 Google 로그인 팝업이 차단됩니다.

2. 카카오 로그인 설정 (KOE101 해결)

01

Kakao Developers 앱 생성

  • developers.kakao.com 접속 → 내 애플리케이션 → 애플리케이션 추가
  • 앱 이름: GALERIE / 회사명 입력
02

JavaScript 키 복사 및 입력

앱 설정 → 앱 키 → JavaScript 키 복사

// docs/index.html 상단 약 41번째 줄
const KAKAO_APP_KEY = "여기에_JavaScript_키_입력";
03

플랫폼 등록 (KOE101 핵심 원인)

앱 설정 → 플랫폼 → Web → 사이트 도메인 등록:

https://bittory.github.io
http://localhost:8100
http://localhost:3000
KOE101 원인: 등록되지 않은 도메인에서 SDK를 초기화하면 발생합니다. 도메인 등록 후 즉시 해결됩니다.
04

카카오 로그인 활성화

  • 제품 설정 → 카카오 로그인 → 활성화 ON
  • Redirect URI 등록: https://bittory.github.io/galerie/
  • 동의항목 → 닉네임, 프로필 사진 체크

3. GitHub 로그인 (선택 — 바로 사용 가능)

01

GitHub OAuth App 생성

  • GitHub → Settings → Developer settings → OAuth Apps → New OAuth App
  • Homepage URL: https://bittory.github.io/galerie/
  • Callback URL: Firebase 콘솔에서 GitHub 공급자 활성화 시 표시되는 URL 사용
02

Firebase에 Client ID/Secret 입력

Firebase 콘솔 → Authentication → Sign-in method → GitHub → Client ID와 Client Secret 입력

4. 최소 작업 요약 (가장 빠른 방법)

Google 로그인만 빠르게 켜려면

  1. Firebase 콘솔 → 새 프로젝트 → 웹 앱 등록
  2. Authentication → Google 활성화 + 지원 이메일 입력
  3. 승인된 도메인에 bittory.github.io 추가
  4. 설정값을 docs/index.html FIREBASE_CONFIG에 붙여넣기
  5. GitHub에 push → 완료
예상 소요 시간: Firebase 신규 설정 기준 약 10분
